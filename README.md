# 🤖 JARVIS - Voice-Controlled Desktop Assistant

JARVIS is a Python-based virtual assistant that responds to your voice commands and automates a wide range of desktop tasks. Inspired by the AI assistant from Iron Man, this project leverages **speech recognition**, **text-to-speech**, **web scraping**, **GUI automation**, and **modular Python programming** to create a functional voice-controlled assistant.

---

## 🧠 Features

- 🎙️ **Voice Recognition & Interaction**  
  Listens and responds using Google's Speech Recognition API and pyttsx3 TTS engine.

- 🗓️ **Task Management & Scheduling**  
  Add, view, and clear daily tasks with voice commands.

- 🌐 **Live Information Fetching**  
  - IPL live scores  
  - Weather updates  
  - Latest news

- 🖥️ **Desktop Automation**  
  - Open/close applications  
  - Control media (play, pause, volume, mute, etc.)  
  - Take screenshots and photos

- 📡 **Internet Speed Test**  
  Check real-time upload and download speed using Speedtest.

- 🔐 **Password Protected Access**  
  Login screen for secure usage.

- 🎯 **Focus Mode**  
  Launch a distraction-free workspace environment.

- 🧠 **Memory Module**  
  Remembers what you tell it and recalls when asked.

- 📩 **WhatsApp Messaging**  
  Sends messages through WhatsApp.
